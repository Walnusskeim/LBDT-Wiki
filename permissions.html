What can I do with software, protected under the LBDT-License?

If you have software in front of you that is protected under the LBDT-License, you are only allowed to do certain things:

You ARE allowed to:

  • Access and view the source code in its unaltered form.

You ARE NOT allowed to:

  • Execute, compile or otherwise run the software
  • Copy, modify or create derivative works based on the software
  • Redistributing the software in whole or in part
  • Using the software, in any form, for commercial, educational, or private purposes
  • Incorporating any portion of the software into other projects, products, or codebases
